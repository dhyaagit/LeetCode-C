# 🔧 C Programming Exercises 
 
This project contains **over than 70 solved exercises** in the C programming language, created by a student to build a strong foundation in programming, data structures, and algorithmic thinking.

These exercises were solved as part of a personal learning journey to master the basics of C, understand algorithmic thinking & how its work, and get hands-on experience with core computer science concepts.

---

## 📚 Topics Covered

It includes a wide range of beginner to intermediate topics:

### 🧠 Programming Fundamentals
- ✅ Variables and Data Types
- 🔁 Loops (`for`, `while`, `do-while`)
- 🔀 Conditional Statements (`if`, `else`, `switch`)
- 📦 Arrays (1D and 2D)
- 📄 Strings
- 🧮 Pointers
- 🔄 Recursion
- 🧩 Functions

### 🧱 Les Structures de Données
- 🧰 `struct` and `typedef struct`
- 🗃️ Linked Lists (basics)
- 📌 Memory management using pointers
- 📚 Stack and Queue (basic usage and implementation)

### 📐 Algorithmic Thinking
- ⚙️ Searching (linear search, binary search)
- 🧹 Sorting (bubble sort, selection sort, etc.)
- 🔢 Mathematical and logic-based problems
- 📊 Problem-solving strategies and algorithm patterns
